// turbo-all

1. Run tsc to check for type errors

```bash
npm run tsc
```

2. Run prettier to fix formatting

```bash
npm run prettier
```

3. Run lint to check for errors

```bash
npm run lint
```

4. Run tests to ensure everything works

```bash
npm run test
```

5. Verify that all steps above succeeded before proceeding.
